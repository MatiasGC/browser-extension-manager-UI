# Frontend Mentor - Browser extensions manager UI

This is my solution to a small personal project where I built a simple Extensions Manager UI.
It allows users to enable/disable extensions, remove them, filter the list, and keep all changes persistent using LocalStorage â€” all built with **HTML**, **CSS**, and **vanilla JavaScript**

## Overview

Users can:

- View a list of browser-like extensions
- Enable or disable each extension
- Remove extensions with a smooth transition animation
- Filter extensions by: All, Enabled, Disabled
- Switch between Light and Dark mode, with theme persistence
- Enjoy a responsive layout
- Keep all extension data stored using LocalStorage (IDs, states, theme)
